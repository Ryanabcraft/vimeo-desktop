Cross-platform

Como funciona
em cada SO.

Windows — Instalador .exe com atalho na área de trabalho, menu iniciar e desinstalador.

Linux — Baixe o vimeo_app.py, dê permissão de execução e rode. Detecta Chrome, Chromium, Brave, Firefox, Opera, Edge e mais.

macOS — Mesmo script Python. Detecta Chrome, Safari, Firefox, Edge, Brave, Opera e Vivaldi no /Applications.
